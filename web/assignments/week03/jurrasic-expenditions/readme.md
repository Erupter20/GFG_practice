# 🦖 Jurassic Expedition Tours – Responsive Booking Website

A modern, responsive travel website for booking immersive expedition tours. Built using HTML, CSS, and Bootstrap with interactive UI components like modals, carousels, and form validation.

---

## 🚀 Features

* Full-screen hero section with background video
* Responsive navigation bar
* Dynamic expedition cards with carousel support
* Booking and details modals
* Image gallery carousel
* Contact form + quick booking system
* Client-side form validation

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* Bootstrap 5

## ⚠️ Limitations

* No backend integration (forms are client-side only)
* No real booking system or database
* Static content (no API)
