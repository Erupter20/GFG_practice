# 🐔 Los Pollos Hermanos – E-commerce UI

A simple static e-commerce website inspired by *Los Pollos Hermanos*, built using HTML and CSS. This project demonstrates basic frontend development skills including layout design, responsive structure, and reusable components.

---

## 🚀 Features

* Homepage with hero section and branding
* Product listing page with grid layout
* Reusable product cards
* Navigation bar with multiple sections
* Clean and simple UI design

---

## 📂 Project Structure

```
project/
│── index.html        # Home page
│── listing.html      # Product listing page
│── css/
│   └── style.css     # Stylesheet
│── assets/           # Images and logos
```

---

## 🛠️ Tech Stack

* HTML5
* CSS3

---

## 📸 Pages Overview

### Home Page

* Branding section with logo and tagline
* Navigation menu
* Hero section with CTA

### Products Page

* Grid-based layout
* Product cards with:

  * Image
  * Title
  * Description
  * Rating
  * Category
  * Price
  * Add to cart button

---

## ⚠️ Limitations

* Static website (no backend or database)
* "Add to cart" button is non-functional
* No responsiveness improvements for all screen sizes

---

## 💡 Future Improvements

* Add JavaScript for cart functionality
